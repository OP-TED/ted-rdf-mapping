Code List Resources for Mappings

JSON and CSV format files can be found at the following location https://github.com/OP-TED/ted-rdf-mapping/tree/main/mappings/package_F03/transformation/resources
The specific URIs are directly used in the technical mappings and can also be found in the conceptual mapping file.

Please ensure you change the path of the link to the tag you are checking.
For example, for tx1.1-delta tag, the links will be:

  • https://github.com/OP-TED/ted-rdf-mapping/tree/tx1.1.delta/mappings/package_F03/transformation/resources

  • https://github.com/OP-TED/ted-rdf-mapping/blob/tx1.1.delta/mappings/package_F03/transformation/conceptual_mappings.xlsx

  • https://github.com/OP-TED/ted-rdf-mapping/tree/tx1.1.delta/mappings/package_F03/transformation/resources
